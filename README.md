# LeetReminders

LeetReminders updates the problem description with a Personal Reminders section with customizable hint boxes you can fill in if you get stuck on a problem and need to look at the solution. Upon revision, you can use these hints to remind yourself of the optimal solution in case you forget.

## Includes:

- Effortless syncing across browsers and devices using secure, username-based serverless storage
- Create, edit, and manage personalized hints with ease
- Smart AI-generated hints that progressively guide users using insights from top community solutions
- Built-in whiteboard for visual reasoning and problem breakdowns
- A clean, intuitive interface that feels native to LeetCode
